# QuickBooks Context for Blog Posts

Reference material for writing QuickBooks-related blog posts.

## Target Audience

- Small business owners
- Bookkeepers
- Accountants supporting QBO clients
- Users migrating between accounting systems

## Writing Style

- Simple, jargon-free language
- Step-by-step instructions
- Include navigation paths
- Note differences between QBO plans (Simple Start, Essentials, Plus, Advanced)
- Screenshots are very helpful

## Product Versions

### QuickBooks Online (QBO)
- Simple Start
- Essentials
- Plus
- Advanced

### QuickBooks Desktop (QBD)
- Pro
- Premier
- Enterprise
- Note: Desktop is being phased out - focus on QBO

## Key Features

### Core Functions
- Chart of Accounts
- Banking connections
- Invoicing
- Bill payment
- Reconciliation
- Basic reporting

### Tracking Features
- Classes (Plus and above)
- Locations (Plus and above)
- Projects (Plus and above)
- Tags

### Automation
- Bank rules
- Recurring transactions
- Invoice reminders

### Reporting
- Profit & Loss
- Balance Sheet
- Cash Flow
- Custom reports (Advanced)

## Common Pain Points

- Limited reporting customization
- Bank feed matching issues
- Class/Location limitations
- Integration challenges
- Export limitations

## Common Tags

```yaml
tags:

```

## Navigation Format

Use ">" for menu paths:
```
Settings > Chart of Accounts > New
```

## Plan Feature Matrix

When discussing features, note plan requirements:
- Simple Start: Basic invoicing, expenses, reports
- Essentials: + Bill management, multiple users
- Plus: + Classes, locations, inventory, projects
- Advanced: + Custom roles, workflows, batch transactions
