# Sage Intacct Context for Blog Posts

Reference material for writing Sage Intacct-related blog posts.

## Target Audience

- Controllers and accounting managers
- CFOs at mid-market companies
- Sage Intacct administrators
- Accountants migrating from QuickBooks/other systems

## Writing Style

- Professional but accessible
- Include navigation paths (e.g., "General Ledger > Reports > Financial Reports")
- Screenshots help significantly (save to `src/assets/`)
- Note permission requirements when relevant
- Reference Sage Intacct documentation where helpful

## Key Concepts

### Core Modules
- General Ledger
- Accounts Payable
- Accounts Receivable
- Cash Management
- Purchasing
- Order Entry
- Inventory (if applicable)

### Dimensional Reporting
- Dimensions (Location, Department, Class, etc.)
- Custom dimensions
- Dimension hierarchies
- Statistical accounts

### Automation Features
- Smart Rules
- Recurring transactions
- Allocation rules
- Workflows and approvals

### Integration
- Sage Intacct API (REST and XML)
- Platform Services
- Marketplace integrations
- Custom integrations

## Common Workflows
