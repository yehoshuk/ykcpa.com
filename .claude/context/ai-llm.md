# AI / LLM Context for Blog Posts

Reference material for writing AI and LLM-related blog posts.

## Target Audience

- Accountants curious about AI
- Professionals exploring AI productivity tools
- Tech-savvy business owners
- People evaluating AI for their workflows

## Writing Style

- Demystify AI concepts
- Practical applications over theory
- Honest about limitations
- Include specific examples and prompts
- Address privacy/security concerns

## Key Topics

### AI Tools
- ChatGPT (OpenAI)
- Claude (Anthropic)
- Gemini (Google)
- Copilot (Microsoft)
- Industry-specific AI tools

### Use Cases for Accountants
- Formula assistance (Excel, Google Sheets)
- Document drafting
- Research and summarization
- Data analysis explanations
- Email composition
- Process documentation

### Prompt Engineering
- Clear, specific instructions
- Providing context
- Few-shot examples
- Iterative refinement
- Role-based prompting

### Limitations to Address
- Numerical reasoning weaknesses
- Hallucinations
- Knowledge cutoff dates
- Context window limits
- Privacy considerations

## Privacy & Security

Always address when discussing AI for business:
- Data handling policies
- What not to share with AI
- Enterprise vs. consumer versions
- Local/offline alternatives
- Compliance considerations (SOC 2, etc.)

## Example Prompts to Include

Format prompts in code blocks:
```
You are an Excel expert. I have a column of dates in A2:A100
and need to extract just the month name. What formula should I use?
```

## Terminology

| Term | Plain Language |
|------|----------------|
| LLM | Large Language Model - the AI "brain" |
| Token | Roughly a word or word-piece |
| Context window | How much text AI can "see" at once |
| Hallucination | When AI makes up false information |
| Fine-tuning | Customizing AI for specific tasks |
| RAG | Retrieval-Augmented Generation - AI + search |

## Balanced Perspective

- Acknowledge both hype and real value
- Be specific about what works and what doesn't
- Update posts as capabilities change
- Note version/date when capabilities matter
